# @zoonk/next

Shared Next.js-specific patterns and utilities for Zoonk applications.

## Installation

This package is included in the monorepo. Add it to your app's dependencies:

```json
{
  "dependencies": {
    "@zoonk/next": "workspace:*"
  }
}
```

## Available Patterns

### Command Palette

A reusable command palette with URL-based search state using [nuqs](https://nuqs.dev) and Next.js parallel routes.

See [Command Palette README](./src/patterns/command-palette/README.md) for detailed usage instructions.
